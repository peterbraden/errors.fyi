---
title: Bluetooth HCI Error Codes
description: "Error codes returned by the Bluetooth Host Controller Interface (HCI), as defined in the Bluetooth Core Specification, Volume 1, Part F. Each code is a single byte value (0x00–0xFF) included in HCI events to indicate the outcome of a command or the reason for a disconnection or failure."
references:
  - https://www.bluetooth.com/specifications/specs/core-specification/
